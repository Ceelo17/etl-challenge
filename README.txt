Challenge desenvolvido em Python no qual estou obtendo contato e me habituando através de um curso sobre desenvolvimento em Python e seus principais frameworks. 
A maior dificuldade foi na parte de teste no qual não não possuo conhecimentos a respeito de testes automatizados, porém seria uma excelente oportunidade para começar a estudar e a respeito do tamanho massivo de dados que consequentemente dificultou na parte de testes também.
O endpoint da API é gerado automáticamente via console Python, porém segue um endpopoint padrão: http://127.0.0.1:5000/, ao acessar a URL será gerado uma mensagem no console na qual mostra que foi iniciado os processos. O tempo de processamento varia em torno de 20 minutos e caso ocorra oscilação na internet será gerado um erro e será necessário reiniciar o processo.
